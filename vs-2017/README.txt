How To Compile:
To compile, open project using the saucer-shoot.sln file found in the vs-2017 folder. 
When it is open, build the program using the Build Solution option under the Build menu on the toolbar. 
After it is built, you can launch the program using the Local Windows Debugger button.

Other Information:
When the game runs, a window will open up and you will be given control of the player.
Use the WASD keys to move left and right and use the space bar to jump
If you walk up to a Cube and press F, the cube will slide away from the player
If you Press the L key, you will load the next level
If you press the K key, you will reload the current level
If you press the J key, you will go back to the first level.
The goal of the game is to get to the blue door which will bring you to the next level.
In the alpha, there is only a handfull of levels each of which demonstrates the functionality of a different mechanic.
